Class Tween

Tweens are timer extensions that upon receiving a 'target' object and its 'property' interpolates it during the timer execution. It will blend the 'from' 'to' values using an EasingFunction or AnimationCurve that maps the [0,1] progress to a new [0,1] actually applying the interpolation.
