Namespace UnityExt.Core.Sys

Classes

Activity

Class that implements any async activity/processing. It can run in different contexts inside unity (mainthread) or separate thread, offering an abstraction layer Monobehaviour/Thread agnostic.

Activity.Manager

Behaviour to handle all activity executions.

Activity<T>

Activity class extension to support unity jobs creation in some methods.

ActivityBehaviour

Component that extends a Monobehaviour so it can execute inside the activity manager context instead of the slow unity loop.

ActivityManager

Class that implements the management of a set of interfaces and activities. It can be used for controlling the execution loop in a different way than the standard one.

Timer

Class that extends an Activity to give support of time tracking, laps and delayed execution start.

IAsyncUpdateable

IJobComponent

Interface that helps job instances to be notified before and after execution to perform data management.

Interfaces for objects that wants to perform late update loops.

IStatusProvider

Interface that provides execution status for managers to handle group of executing nodes.

Interface that provides execution status for managers to handle group of executing nodes.

IThreadUpdateable

Interfaces for objects that wants to perform update loops inside a thread

IUpdateable

Interfaces for objects that wants to perform update loops.
